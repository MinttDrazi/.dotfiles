# .dotfiles

All my .config files for MacOS

## Ghostty

1. Install Ghostty: [link](https://ghostty.org/)
2. Copy Ghostty config file to the directory

## Neofetch

1. Install Neofetch: [link](https://github.com/dylanaraps/neofetch)
2. Copy Neofetch config file to the directory

## Starship

1. Install starship: [link](https://starship.rs/)
2. Copy starship config file to the directory
3. Add these lines to `.zshrc`

   ```
   export STARSHIP_CONFIG=~/.config/starship/starship.toml
   eval "$(starship init zsh)"
   ```
